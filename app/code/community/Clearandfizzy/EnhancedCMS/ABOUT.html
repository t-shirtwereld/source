Export and Import CMS Pages & Static Blocks between servers via CSV files

About Enhanced CMS

Often our Content Management teams need to develop and test their content on a staging or local server before synchronising their changes across to our production database. Using this extension you can export and import CMS Pages & Static Blocks from one server to another via CSV Files.

Key Features

* Export / Import CMS > Static Blocks as a CSV File.
* Export / Import CMS > Pages as a CSV File.

Support and Guarantee

This is a FREE plugin and comes with limited email support.

Please note that this extension is provided as is. It is recommended to always backup your installation prior to use.
